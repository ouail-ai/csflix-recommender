# csflix-recommender
csflix-recommender is an interactive web platform that enables users to rate movies and get personalized recommendations based on their ratings.

You will need to have a valid TMDB API key in order to fetch movie data.

## Backend

### Project setup

```
cd backend
npm install
cp .env.example .env
```

### Run database migrations

```
npm run migration:run
```

### Start and auto-reload for development

```
npm run dev
```

### Start for production

```
npm run start
```

### Lint and fix files

```
npm run lint
```
